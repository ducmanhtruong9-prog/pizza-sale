# pizza-sale
Designed and built an end-to-end interactive Power BI dashboard analysing 48,620 pizza sales transactions across 2015, covering data transformation, modelling, DAX analytics, and business intelligence reporting.

Project Overview

This project analyses pizza sales performance using Power BI to identify:
- Best-selling pizzas
- Revenue trends
- Customer purchasing patterns
- Peak ordering hours
- Category performance

Tools Used: Power BI, SQL Server, Excel

Key Insights
- Classic pizzas generated the highest revenue.
- Friday evenings had the highest order volume.
- Large-sized pizzas contributed over 45% of revenue.
- Business Recommendations
- Increase inventory before weekend peaks.
- Promote underperforming pizza categories.
- Bundle high-margin products.
